1. Procedure:
   A procedure is a collection of SQL statements.

 Stored Procedure Features:-
 1.Stored Procedure increases the performance of the applications.
 2.tored procedure reduce the time.

How to create a procedure:-
  CREATE PROCEDURE procedure_name[parameter]

procedure contain IN para,OUT para and INOUT para.

How to call a stored procedure?
  CALL procedure_name ( parameter(s)) 
Example:  

       DELIMITER //

CREATE PROCEDURE himamshu(IN p_id INT)
BEGIN
    SELECT 
        *
    FROM
        ehat_treatment
    WHERE
        patient_id = p_id;
END //

DELIMITER ;

call himamshu(12)
